# Internet Archive

A [Raycast](https://raycast.com/?via=73820f) extension for searching content from the [Internet Archive](https://archive.org/).

## Search Pages

1. Open the `Search Pages` command in Raycast
2. Enter a URL such as `http://www.gzevergrandefc.com/photos.aspx`
3. Browse the results and open the snapshot you need
